# Examples

| Example | Description |
|---|---|
| [github_runners_pat](./github_runners_pat/) | GitHub runners with PAT authentication in ALZ Corp |
| [github_runners_app_auth](./github_runners_app_auth/) | GitHub runners with GitHub App authentication |
| [azuredevops_agents_pat](./azuredevops_agents_pat/) | Azure DevOps agents with PAT authentication |
| [azuredevops_agents_uami](./azuredevops_agents_uami/) | Azure DevOps agents with UAMI (no secrets) |

All examples expect subnet IDs and (optionally) DNS zone IDs from the ALZ Vending Module.
See [FIREWALL-RULES.md](../FIREWALL-RULES.md) for required firewall openings.
